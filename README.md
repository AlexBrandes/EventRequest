# EventRequest
A backend server in NodeJs
Includes:
- Body parsers 
1) Form Body Parser
2) Multipart Body Parser 
3) Json Body Parser
- Cookie parser
- Session security
- File streams
- Templating engine
- Easy Routing
- Middlewares
- Logging
1) Console
2) File
- Static Routes
- Request timeout
- Error handling
- Caching ( in memory )
